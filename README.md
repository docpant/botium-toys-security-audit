# Botium Toys Security Audit

## Overview
This project presents an internal IT audit for Botium Toys, a fictional company, using the NIST Cybersecurity Framework (CSF).

## Objective
To identify security risks, evaluate existing controls, and assess compliance with industry standards.

## Framework Used
- NIST Cybersecurity Framework (CSF)

## Key Findings
- Lack of access control (no least privilege)
- No encryption for sensitive data
- No backups or disaster recovery plan
- Missing intrusion detection system (IDS)

## Project Structure
- audit-scope.md
- risk-assessment.md
- controls-assessment.md
- compliance-assessment.md
- recommendations.md

## Outcome
This audit identifies critical security gaps and provides recommendations to improve the organization’s security posture.
